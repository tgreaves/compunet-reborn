USER GUIDE

Compunet is an interactive computer network for the Commodore 64. This guide covers navigation, uploading, mail, and the commands available to you.

LOGGING ON

  1. Load and run the client (see Connect page)
  2. Type CONNECT at the ready prompt
  3. Enter your User ID and password (press RETURN after each)
  4. The system sends terminal software (“LINKING”) — this takes a few seconds
  5. A personal information screen appears with the directory duckshoot

THE DUCKSHOOT

The duckshoot is the command menu at the bottom of the screen. Use cursor left/right to scroll through available commands, then press RETURN to select.

Different commands are available depending on where you are in the system (directory, editor, mail, etc.).

DIRECTORIES

Compunet is organised as directories arranged in a tree structure. Each entry may contain text to read, a program to download, and/or a further sub-directory.

DIRECTORY ENTRIES

Each entry shows a letter code indicating its type:

  • T — Text pages (use SHOW to read)
  • P — Program (use BUY to download)
  • D — Directory-only entry (use DIR to enter)
  • + after the type — has a sub-directory beneath it

A number after the letter indicates how many pages/KB the entry contains.

Use cursor up/down to highlight an entry with a blue bar. The columns show PRICE, LIFE (days remaining), AUTHOR, and VOTE score — toggle F7/F8 to see all columns.

GETTING AROUND

DIRGo deeper — view the sub-directory beneath a highlighted entry
BACKGo up — return to the parent directory
GOTOJump directly to a page number from anywhere
SHOWRead text pages of the highlighted entry
BUYDownload a program or view priced content

READING PAGES

Highlight a text entry (T) and select SHOW. The first page appears. If there are multiple pages:

  • MORE — next page
  • ALL — show all pages rapidly
  • FINISH — return to directory

DOWNLOADING PROGRAMS

  1. Ensure your disk drive is connected and switched on
  2. Highlight the program entry (P) and select BUY
  3. Downloading takes about 10 seconds per KB
  4. When complete, you are asked SAVE FILENAME?
  5. Press RETURN to save under the existing name

COURIER (ELECTRONIC MAIL)

Courier is Compunet's mail system. If you have unread mail, a red pillarbox appears on your personal information page at login, or the word MAIL appears in red on the next directory you visit.

READING MAIL

  1. Select MAIL from the duckshoot
  2. Highlight the letter and select SHOW
  3. Read mail is automatically removed after 2 days

SENDING MAIL

  1. Have your message ready in the Editor
  2. Select MAIL, then SEND
  3. Type the title of your letter
  4. Type the recipient's User ID (up to 5 recipients)
  5. Press RETURN twice after the last ID
  6. Find your page in the Editor and select SEND
  7. Select FINISH when done

THE JUNGLE

The Jungle is where users upload their own content — text, graphics, and programs. To upload:

  1. Navigate to an appropriate directory with space
  2. Select UPLD
  3. Enter: title, type (T/P), price (0 for free), and LIFE (days to keep it online)
  4. For text: find your page in the Editor and select SEND
  5. For programs: use SEND (if already in memory) or LOAD first
  6. Select FINISH when done

PARTYLINE

Partyline is Compunet's live multi-user chat. Access it from the main directory. Type your message and press RETURN twice to send. Commands start with *:

  • *help — show help
  • *who — list users online
  • *alias NAME — set a nickname
  • *enter ROOM — join/create a room
  • *dice N — roll N-sided dice
  • *quit — leave partyline

THE EDITOR

The Editor is a full-screen page creator with Commodore graphics and colours. It can be used both on-line (select EDITR from the directory duckshoot) and off-line (type EDITOR at the BASIC prompt before connecting).

Every page you SHOW on Compunet is automatically stored in your Editor, so you can re-read, save or print it later. The Editor holds 10–15 pages; as more go in, earlier pages are lost unless saved.

OFFLINE EDITOR

To use the Editor off-line, type EDITOR (or SHIFT+E) at the BASIC prompt. This gives a white page with a blue border and duckshoot. Prepare pages off-line, then connect and upload them — saves time online.

TIPS

KEYBOARD SHORTCUTS

  • F7/F8 — Toggle directory column display (Price, Life, Author, Vote)
  • Cursor Up/Down — Highlight directory entries
  • Cursor Left/Right — Scroll duckshoot commands
  • RETURN — Execute highlighted command
  • RUN/STOP — In Partyline, sends *quit
