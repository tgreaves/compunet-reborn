HOW TO CONNECT

Compunet Reborn runs on a real (or emulated) Commodore 64 with a SwiftLink-compatible serial interface.

SUPPORTED HARDWARE

  • VICE C64 emulator (x64sc) — download here
  • C64 Ultimate (Ultimate 64, Ultimate-II+) — uses the built-in SwiftLink-to-TCP bridge
  • Real C64 + SwiftLink with a WiFi modem

VICE SETUP

  1. Go to Settings → Peripheral Devices → RS232
  2. Enable ACIA RS232 interface emulation
  3. Set Device to Serial 3
  4. In the RS232 devices section for Serial 3, set:
    • Host: vme.compunet.live:6400
    • Baud: 1200
    • IP232: Unchecked

C64 ULTIMATE SETUP

  1. Configure: Modem Interface → ACIA / SwiftLink, DE00/NMI

CONNECTING (D64)

  1. Attach the D64 to drive 8
  2. Type LOAD "COMPUNET",8
  3. Type RUN
  4. Type CONNECT
  5. Enter your User ID and password
  6. The duckshoot menu appears — you're in!

CONNECTING (PRG)

  1. In VICE: drag the PRG file into the emulator window, or use File → Smart Attach
  2. On C64 Ultimate: select the PRG in the file browser and choose Run
  3. Type CONNECT
  4. Enter your User ID and password
  5. The duckshoot menu appears — you're in!

TIPS

NAVIGATION

Use the duckshoot (bottom menu) to navigate. Cursor left/right scrolls commands. RETURN selects. F7/F8 cycles directory columns.
