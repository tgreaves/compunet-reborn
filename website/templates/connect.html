HOW TO CONNECT

There are two ways to connect to Compunet Reborn:

  • PETSCII Terminal (easiest) — use SyncTerm or any PETSCII terminal program
  • Custom C64 Client — full experience on a real or emulated Commodore 64

STEP 1: REGISTER

You need a registered account to connect. Registration is free.

Register for an account

Once approved, you'll receive your User ID and can set your password.

OPTION A: PETSCII TERMINAL (SyncTerm)

The quickest way to connect. Works on Windows, Mac, and Linux with no C64 hardware or emulator required.

1. Download SyncTerm

Download from syncterm.bbsdev.net and install.

2. Create a Connection Entry

Create a new directory entry in SyncTerm with these settings:

NameCompunet Reborn
Addressvme.compunet.live
Connection TypeRaw
TCP Port6401
Screen ModeC64
FontCommodore 64 (UPPER)
Fake Comm RateYour choice on how retro you want to be!

3. Connect

  1. Select your Compunet Reborn entry and connect
  2. You should see a PETSCII block character confirming your terminal is configured correctly
  3. Enter your User ID and password
  4. The welcome screen appears — you're in!
Tip: Ensure your terminal is set to 40 columns × 25 rows. If the display looks garbled, check your Font and Screen Mode settings.

File Transfers

Uploads and downloads use XMODEM-CRC (also supports XMODEM-1K). Use your terminal's XMODEM send/receive function to transfer files.

Other Compatible Terminals

Any terminal program with PETSCII support and XMODEM-CRC file transfer will work. On a C64 or C64 Ultimate, the following terminal programs can connect to port 6401:

  • CCGMS — classic C64 terminal with PETSCII and XMODEM support
  • StrikeTerm — modern C64 terminal program
  • UltimateTerm — designed for the C64 Ultimate

Configure your C64 terminal to connect to vme.compunet.live:6401 using Raw TCP (no Telnet).

OPTION B: CUSTOM C64 CLIENT

The full Compunet experience on a real (or emulated) Commodore 64 with SwiftLink-compatible serial interface.

Supported Hardware

  • VICE C64 emulator (x64sc) — download here
  • C64 Ultimate (Ultimate 64, Ultimate-II+) — built-in SwiftLink-to-TCP bridge
  • Real C64 + SwiftLink with a WiFi modem

Download Client

compunet-reborn-live.prg Auto-connects to Compunet (recommended)
compunet-reborn-sfx.prg Manual connection (enter server address when prompted)
⚠ WINDOWS VICE USERS: If you experience connection issues with the auto-connect client, use compunet-reborn-sfx.prg instead. Enter vme.compunet.live:6400 at the phone number prompt.

Setup

VICE

  1. Go to Settings → Peripheral Devices → RS232
  2. Enable ACIA RS232 interface emulation
  3. Set Device to Serial 3
  4. In the RS232 devices section for Serial 3, set:
    • Host: vme.compunet.live:6400
    • Baud: 1200
    • IP232: Unchecked

C64 ULTIMATE

  1. Go to the Ultimate menu
  2. Configure: Modem Interface → ACIA / SwiftLink, DE00/NMI
  3. Ensure Command Interface is Enabled

Connecting

  1. In VICE: drag the PRG file into the emulator window, or use File → Smart Attach. If VICE does not RUN automatically (with flashing bars showing the decompression), type RUN.
  2. On C64 Ultimate: select the PRG in the file browser and choose Run
  3. Wait for the client to finish decompressing (flashing bars visible)
  4. Type CONNECT
  5. (If using manual SFX client): Enter vme.compunet.live:6400
  6. Enter your User ID and password
  7. The welcome screen appears — you're in!
