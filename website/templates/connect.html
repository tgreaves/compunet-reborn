HOW TO CONNECT

Compunet Reborn runs on a real (or emulated) Commodore 64 with a SwiftLink-compatible serial interface. The easiest way to get started is with the VICE emulator.

PREREQUISITES

  1. VICE C64 emulator (x64sc) — download here
  2. The Compunet Reborn D64 disk image — download here
  3. A registered account — register here

VICE SETUP

  1. Go to Settings → Peripheral Devices → RS232
  2. Enable ACIA RS232 interface emulation
  3. Set Device to Serial 3
  4. In the RS232 devices section for Serial 3, set:
    • Host: vme.compunet.live:6400
    • Baud: 1200
    • IP232: Unchecked
  5. Attach compunet-reborn.d64 to drive 8

CONNECTING

  1. Load the client: LOAD "COMPUNET",8,1
  2. Type SYS 33184
  3. Type CONNECT
  4. Enter any number at the phone number prompt (the connection is configured in VICE settings)
  5. Enter your User ID and password
  6. The duckshoot menu appears — you're in!

TIPS

NAVIGATION

Use the duckshoot (bottom menu) to navigate. Cursor left/right scrolls commands. RETURN selects. F7/F8 cycles directory columns.
