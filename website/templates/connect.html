HOW TO CONNECT

Quickest start: Download compunet-reborn-live.crt, attach in VICE or C64 Ultimate, power on. Register first if you haven't already.

STEP 1: REGISTER

You need a registered account to connect. Registration is free.

Register for an account

STEP 2: CHOOSE YOUR CLIENT

💾 VICE / C64 Ultimate

Cartridge image — just attach and boot.

compunet-reborn-live.crt Auto-connect

💻 Real C64 + SwiftLink

PRG file — LOAD then RUN from disk or SD card.

STEP 3: SETUP

VICE Setup (click to expand)
  1. Download VICE from vice-emu.sourceforge.io
  2. Go to Settings → Cartridge/IO Settings → RS-232 interface emulation
  3. Enable ACIA RS232 interface emulation
  4. Set Device to Serial 3
  5. In the RS232 devices section for Serial 3:
    • Host: vme.compunet.live:6400
    • Baud: 1200
    • IP232: Unchecked
  6. Attach the CRT: File → Attach cartridge image → select the .crt file
  7. Reset the emulator (Machine → Reset → Hard)
C64 Ultimate Setup (click to expand)
  1. Copy the CRT file to your USB or SD storage
  2. In the Ultimate menu: Configure → Modem Interface → ACIA / SwiftLink, DE00/NMI
  3. Ensure Command Interface is Enabled
  4. Navigate to the CRT file and select it as a cartridge
  5. Reset or power cycle
SyncTerm Setup (click to expand)

Real C64 + SwiftLink / WiFi Modem (click to expand)
  1. Copy the PRG file to a D64 image or SD card
  2. LOAD "COMPUNET-REBORN-LIVE",8
  3. RUN
  4. The client decompresses and boots to the Compunet prompt
  5. Type CONNECT

For the manual version, enter vme.compunet.live:6400 at the phone number prompt.

STEP 4: CONNECT

  1. The system boots to the Compunet BASIC prompt.
  2. Type CONNECT (CRT/PRG) or select your entry (SyncTerm).
  3. For CRT/PRG clients: LINKING downloads the terminal software (~3 seconds on first connect).
  4. Enter your User ID and password.
  5. The welcome screen appears — you're in!
⚠ Windows VICE: If the auto-connect CRT fails to connect, use the manual version and enter vme.compunet.live:6400 at the phone number prompt.
